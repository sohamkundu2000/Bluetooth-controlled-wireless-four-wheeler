# Bluetooth-controlled-wireless-four-wheeler

Component used-

   1)Arduino Uno
   2)Hc05 Bluetooth module
   3)H-bridge motor driver
   4) 4-dc motor with gear
   5) jumper wires
   6) Lithium ion battery
   7) Led
